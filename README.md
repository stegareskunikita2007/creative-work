# StudBot — бот-модератор студенческих чатов ВКонтакте

## Описание

Бот подключается к групповым чатам ВКонтакте, анализирует сообщения и,
если студент просит о помощи, рекомендует подходящий курс с платформы Stepik.

### Возможности

- Подключение к VK API через Long Poll
- Загрузка курсов с Stepik по 25+ техническим темам
- Лемматизация русских слов (MyStem от Яндекса)
- Исправление опечаток (SymSpell, 108 000 слов)
- Поиск курсов по смысловой близости (TF‑IDF + косинусное сходство)
- Сохранение токена между запусками
- Цветные логи с фильтрацией

### Технологии

| Компонент | Технология |
|-----------|-----------|
| Язык | C++17 |
| GUI | Qt 6.11 |
| VK API | Bots Long Poll API |
| Курсы | Stepik REST API |
| Лемматизация | MyStem 3.1 (Яндекс) |
| Исправление опечаток | SymSpell + QProcess |
| Поиск по смыслу | TF‑IDF + Cosine Similarity |
| Сборка | qmake / Qt Creator |

### 🔧 Сборка и запуск

1. Установите Qt 6.5+ с компонентом Qt Widgets.
2. Склонируйте репозиторий.
3. Откройте файл StudBot.pro в Qt Creator, выберите комплект сборки и соберите проект сочетанием клавиш Ctrl+B.
4. Скачайте утилиту MyStem с официального сайта Яндекса по прямой ссылке и поместите файл mystem.exe в папку сборки туда же, где лежит исполняемый файл программы.
https://yandex.ru/dev/mystem/
5. Найдите в папке spellchecker `spellchecker_cli.exe` и в spellchecker/data - `frequency_dictionary_ru.txt` и оба файла переместите в папку с исполняемым файлом.
6. Запустите программу, вставьте токен VK API, полученный в настройках вашего сообщества ВКонтакте, и нажмите кнопку «Запустить». Бот начнёт загрузку курсов и будет готов к работе.
7. Получение VK API и настройка бота для работы: создаём своё сообщество VK, переходим в управление -> на боковой панели справа находим "Дополнительно" и нажимам -> далее переходим на этой же боковой панели в раздел "Работа с API" и нажимаем создать ключ (предоставляем доступ приложению к управлению сообществом и к сообщениям сообщества). Этот API ключ нужно будет вставить в приложение StudBot один раз и он там сохранится. Далее нужно включить ещё Long Poll API (он находится там же, где получали ключ API, но в другой вкладке), в Long Poll API в типах событий нужно поставить галочку для "Входящее сообщение".
8. Можно писать в личные сообщения сообщества или создать групповой чат и добавить туда этого бота (сообщество). И так, и так бот будет отвечать на просьбы помощи по темам, которые будут интересны студентам технических вузов.
9. Добавление бота в групповой чат: переходим снова в управление сообщества -> сообщения -> настройки для бота (включаем) -> ставим галочку "Разрешать добавлять сообщество в чаты". В меню сообщества появится "Добавить в чат".
