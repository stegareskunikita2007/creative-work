# StudBot — бот-модератор студенческих чатов ВКонтакте

## Описание

Бот подключается к групповым чатам ВКонтакте, анализирует сообщения и,
если студент просит о помощи, рекомендует подходящий курс с платформы Stepik.

### Возможности

- Подключение к VK API через Long Poll
- Загрузка курсов с Stepik по 25+ техническим темам
- Лемматизация русских слов (MyStem от Яндекса)
- Исправление опечаток (SymSpell, 108 000 слов)
- Поиск курсов по смысловой близости (TF‑IDF + косинусное сходство)
- Сохранение токена между запусками
- Цветные логи с фильтрацией

### Технологии

| Компонент | Технология |
|-----------|-----------|
| Язык | C++17 |
| GUI | Qt 6.11 |
| VK API | Bots Long Poll API |
| Курсы | Stepik REST API |
| Лемматизация | MyStem 3.1 (Яндекс) |
| Исправление опечаток | SymSpell + QProcess |
| Поиск по смыслу | TF‑IDF + Cosine Similarity |
| Сборка | qmake / Qt Creator |

### 🔧 Сборка и запуск

1. Установите Qt 6.5+ с компонентом Qt Widgets.
2. Склонируйте репозиторий.
3. Откройте файл StudBot.pro в Qt Creator, выберите комплект сборки и соберите проект сочетанием клавиш Ctrl+B.
4. Скачайте утилиту MyStem с официального сайта Яндекса по прямой ссылке и поместите файл mystem.exe в папку сборки туда же, где лежит исполняемый файл программы.
https://yandex.ru/dev/mystem/
5. Запустите программу, вставьте токен VK API, полученный в настройках вашего сообщества ВКонтакте, и нажмите кнопку «Запустить». Бот начнёт загрузку курсов и будет готов к работе.
